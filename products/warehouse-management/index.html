Licensing

Distribution and implementation of software products containing the configuration of "Warehouse management ITVectura WM" are carried out by licensed partners of IT VECTURA LLC. There are no mandatory requirements for certified specialists.

The number of licenses purchased for the use of the "Warehouse management ITVectura WM" configuration is determined based on the need:

— maximum number of concurrent users using this configuration

— maximum number of concurrent mobile clients using this configuration

— maximum number of unique registered units of power in the loading and unloading zone in the system

"The expansion of the number of automated workstations is carried out by purchasing client licenses for the "Warehouse management ITVectura WM" platform for:

— 1, 5, 10, 20, 50, 100, 300, 500 workstations

— 1, 5, 10, 20, 50, 100, 300, 500 mobile clients

— 1, 5, 10, 20, 50, 100, 300, 500 unique vehicles used during 1 calendar year

— 1, 5, 10, 20, 50, 100, 300, 500 workstations for unique units of power in the loading and unloading zone

The acquisition of software products containing the configuration of 'Warehouse management ITVectura WM' does not include licenses and cannot be used without acquiring licenses, the number of which is calculated upon purchase based on the need for the maximum number of simultaneously working users, simultaneously working mobile clients, the number of unique registered vehicles in the system, and the number of unique registered units of loading and unloading zone power.
